## Установка
Сервер:<br>
```javascript
npm install
```
```javascript
npm run dev
```
Клиент:<br>
```javascript
npm install
```
```javascript
npm start
```

## Стек технологий
- frontend: React + MobX + React Bootstrap;
- backend: NodeJS + Express.js + PostgreSQL + Sequelize + JWT;
